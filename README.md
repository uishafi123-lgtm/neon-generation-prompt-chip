# 🌃 Neon Generation Prompt Chip

Generates neon/cyberpunk style art prompts with glowing electric effects for AI image generators like Midjourney, Stable Diffusion, and DALL-E.

## ✨ Features

- Cyberpunk and futuristic subjects
- Electric neon color effects
- Glowing particles and energy aura
- Dark cyberpunk backgrounds
- High-quality art prompts

## 🚀 Quick Start

```bash
pip install -r requirements.txt
python prompt_generator.py
